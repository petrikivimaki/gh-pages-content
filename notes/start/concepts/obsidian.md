# Obsidian in Brief

[Obsidian](https://obsidian.md/) is a local-first Markdown notes app. A collection of notes is called a vault, and each note is a normal file on disk. That makes Obsidian a good authoring tool for Publisher: the same files can be read locally, versioned with Git, and published on the web.

Useful Obsidian ideas:

- Notes are Markdown files
- Folders can be organized however you like
- `[[wiki links]]` connect notes
- Frontmatter stores metadata at the top of a file
- Plugins can add authoring features while the underlying files stay portable

Publisher supports enough Obsidian-style conventions to make a vault feel familiar online. See [[reference/extensions/obsidian-and-publisher]] for the short list.
